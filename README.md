## ūüßģ Formulas Used

### Forex
```
Pips     = |Exit ‚ąí Entry| √∑ Pip Size
Points   = Pips √ó 10
P&L      = Pips √ó Pip Value √ó Lots
Risk %   = (SL Pips √ó Pip Value √ó Lots) √∑ Account Balance √ó 100
Safe Lot = (Account Balance √ó 0.02) √∑ (SL Pips √ó Pip Value)
R:R      = TP Pips √∑ SL Pips
```

### Crypto
```
P&L           = (Exit ‚ąí Entry) √ó Amount √ó Leverage
Margin Used   = Entry √ó Amount √∑ Leverage
Liq. Price    = Entry √ó (1 ‚ąí 1/Leverage)   [BUY]
              = Entry √ó (1 + 1/Leverage)   [SELL]
Risk %        = SL Loss √∑ Account Balance √ó 100
Safe Amount   = (Account Balance √ó 0.02) √∑ (SL Distance √ó Leverage)
```

## ‚öôÔłŹ How to Use

### Forex Tab
1. Enter your **Account Balance** at the top
2. Select your **Currency Pair**
3. Choose **BUY** or **SELL**
4. Enter **Entry Price**, **Exit Price**, **Stop Loss**, **Take Profit**
5. Select or type your **Lot Size**
6. Choose **Account Currency**
7. Click **CALCULATE** ‚ÄĒ or just type, results update live

### Crypto Tab
1. Enter your **Account Balance**
2. Select your **Crypto Pair** ‚ÄĒ live price auto-fills entry
3. Choose **BUY** or **SELL**
4. Enter **Entry Price** and **Exit Price**
5. Set **Stop Loss** and **Take Profit**
6. Enter **Coin Amount**
7. Drag the **Leverage Slider** ‚ÄĒ 1x means Spot trading
8. Risk warnings and results update instantly
